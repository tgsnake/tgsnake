## Overview
Forwards messages by their IDs.
## Parameters
```ts 
ctx.telegram.forwardMessages(chatId,fromChatId,messageId,more)
```
- `chatId` (_number|string_)
    - Destination.
- `fromChatId` (_number|string_)
    - Source of messages.
- `messageId` (_number[]_)
    - IDs of messages which will forwarded.
- `more` (_object_)
    - `withMyScore` (_boolean_)
        - When forwarding games, whether to include your score in the game.
    - `silent` (_boolean_)
        - Whether to send messages silently (no notification will be triggered on the destination clients)
    - `background` (_boolean_)
        - Whether to send the message in background.
    - `scheduleDate` (_number_)
        - Scheduled message date for scheduled messages.
## Example
```ts headlight=2
bot.command("forward", async (ctx) => {
    let results = await ctx.telegram.forwardMessages(ctx.chat.id,ctx.chat.id,[ctx.id]) 
    return console.log(results)
})
```
