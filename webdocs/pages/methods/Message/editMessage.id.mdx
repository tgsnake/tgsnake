## Gambaran
Metode ini memungkinkan Anda untuk mengedit pesan.
## Parameter
```ts
ctx.telegram.editMessage(chatId,messageId,text, more)
```
- `chatId` (_number|string|bigint_)
    - Obrolan/Grup/Id Saluran.
- `messageId` (_number_)
    - Id pesan yang akan diedit.
- `text` (_string_)
    - Pesan/Keterangan Baru. Anda dapat meneruskan dengan string kosong (`""`) jika Anda ingin mengedit media.
- `more` (_objek_)
    - `noWebPage` (_boolean_)
        - Nonaktifkan pratinjau halaman web
    - `media` (_[Api.TypeInputMedia](https://gram.js.org/beta/modules/tl_api.api.html#typeinputmedia)_)
        - Media terlampir baru
    - `replyMarkup` (_TypeReplyMarkup_)
        - Markup balasan untuk keyboard sebaris.
    - `entities` (_Entities[]_)
        - Entitas Pesan
    - `scheduleDate` (_number_)
        - Tanggal pesan terjadwal untuk pesan terjadwal.
    - `parseMode` (_string_)
        - Modus Parsing.
