## Gambaran
Ubah gambar profil obrolan.
## Parameter
```ts
ctx.telegram.editFoto(chatId,foto)
```
- `chatId` (_number|string|bigint_)
    - Obrolan/Grup/Channel yang akan mengubah gambar profil.
- `foto` (_string|Penyangga_)
    - Lokasi file berada/Url/Buffer file.
## Contoh
```ts highlight=3
bot.command("editPhoto", async (ctx)=>{
    if(!ctx.chat.private){
        let hasil = await ctx.telegram.editPhoto(ctx.chat.id,"https://tgsnake.js.org/images/tgsnake.jpg")
        console.log(hasil)
    }
})
```
