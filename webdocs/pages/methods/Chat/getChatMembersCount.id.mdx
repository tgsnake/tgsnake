## Overview
Get the number of members in a chat.

## Parameters
```ts 
ctx.telegram.getChatMembersCount(chatId)
```

- `chat` (_number|string_)
    - Chat or channels id to getting the number of members.

## Example
```ts headlight=2
bot.command("getChatMembersCount",async (ctx) => {
    let results = await ctx.telegram.getChatMembersCount(ctx.chat.id)
    console.log(results)
})
```
