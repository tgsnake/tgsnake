## Overview 
Change the profile picture of chat. 
## Parameters 
```ts 
ctx.telegram.editPhoto(chatId,photo)
```
- `chatId` (_number|string_) 
    - Chat/Groups/Channel which will change the profile picture. 
- `photo` (_string|Buffer_) 
    - The location where the file is located/Url/The buffer of the file.
## Example 
```ts:index.ts {3} showLineNumber 
bot.command("editPhoto", async (ctx)=>{
    if(!ctx.chat.private){
        let results = await ctx.telegram.editPhoto(ctx.chat.id,"https://tgsnake.js.org/images/tgsnake.jpg") 
        console.log(results)
    }
})
```
